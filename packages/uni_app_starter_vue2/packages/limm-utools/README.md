# utools

适合 uni-app 开发者的工具包。

## 📌 快速上手

`npm`

```shell
npm i @limm/utools
# or
yarn add @limm/utools
```

`uni_modules`

使用 `HbuilderX` 从插件市场导入即可。

插件市场地址: [https://ext.dcloud.net.cn/plugin?name=limm-utools](https://ext.dcloud.net.cn/plugin?name=limm-utools)

## 📌 文档
