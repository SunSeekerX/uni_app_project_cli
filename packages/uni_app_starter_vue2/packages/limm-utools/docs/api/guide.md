### 使用指南

全局导出的对象

```javascript
export { dayjs, uuid, CommonUtil, ValidateUtil, EncodeUtil, EncryptUtil, PerformanceUtil, RandomUtil }
```

- dayjs - 同 dayjs 库对象
- uuid - 同 uuid 库对象
- CommonUtil - 基础方法
- ValidateUtil - 验证方法
- EncodeUtil - 编码方法
- EncryptUtil - 加密方法
- PerformanceUtil - 性能测试方法
- RandomUtil - 随机发生工具
